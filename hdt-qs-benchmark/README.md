## Dataset size in function of the BSBM product count

- `10000` = `3,5M`
- `50000` = `17M`
- `100000` = `34M`
- `200000` = `69M`
- `500000` = `173M`
- `1000000` = `346M`
- `2000000` = `692,62M`
- `5000000` = `1.7B`
